Bigmart Data Pipeline: Pipeline to migrate data from bigmart server to Elasticsearch, Cassandra and mongodb

Date: Aug 2021- Feb 2022

I collaborated up with team of developers and data scientists to build data pipeline to ingest data from bigmart server, analyze, filter and manage it into elasticsearch and cassandra for further analysis and visualization of insights.

Overview

We built a java-scala application to fetch data from server. These data were analysed, processed, aggregated and filtered using spark and then inserted into elasticsearch, cassandra and mongodb. Now, they are used for the further analysis and visulization into web and mobile applications.
