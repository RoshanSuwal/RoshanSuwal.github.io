About

16 May 2022

Apostle provides an analytic dashboard to view API usage within various time interval as well as error code analysis.

Currently, supported Analysis are :

  1. Allocated calls based usage

    1. usage out of paid calls

    2. usage out of grace calls

  2. Time interval based usage

    1. Hourly Calls

    2. Daily calls

    3. Monthly calls

    4. Yearly calls

  3. Api calls graph with support of date calendar

  4. Graph for duration of backend response [ DoBR]

  5. Pie chart for all occurred status code

  6. Horizontal bar chart for status code with date calendar
