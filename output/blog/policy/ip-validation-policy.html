IP Validation

16 May 2022

IP Validation validate the Ip address of the request. Only the requests from the valid IP source can proceed further

Classification of IP Addresses

1. Marked IP Address

Marked IP Address are the suspicious IP address that has performed some miscellaneous activities. These Ip address are closely monitored by the system and if further miscellaneous activities are performed, then it is blocked. These address can further send requests.

2. Blocked IP Address

When the suspicious/marked ip address further perform miscellaneous or illegal activities, then there address are marked as block ip address. They cannot further send requests to the system.

3. Valid IP Address

These are the Ip address from the valid sources. They are allowed to send requests to system.

How Validation is done?

All the blocked and marked Ip address are stores in BLOCKED_KEYSPACE AND MARKED_KEYSPACE in hash data structure in redis database. The system first checks if the given ip address is in BLOCKED_KEYSPACE, if present it returns the IPException else proceed forward.

A set of rules has been implemented which is responsible for analysing the ip address. It registers the IP address in MARKED_KEYSPACE and BLOCKED_KEYSPACE.
